python 新建文件夹

os.mkdir() 和 os.makedirs()

  • 使用 os.mkdir("log/flylog"),报错如下:

    FileNotFoundError: [WinError 3] 系统找不到指定的路径

  • 发现是mkdir()函数的问题，换成os.makedirs()就ok了。

  • 原因是因为os.mkdir()只能创建一级目录。而os.makedirs()可以创建多级目录。
